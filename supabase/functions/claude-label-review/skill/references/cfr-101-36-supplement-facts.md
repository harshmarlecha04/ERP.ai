# Supplement Facts Panel — 21 CFR 101.36

This is the deep-dive reference for Supplement Facts panel review. Load this when the label has a Supplement Facts panel (i.e., always, for dietary supplements).

## 1. Title

**"Supplement Facts"** — must appear as the panel heading.

- In bold type.
- The **largest type size** in the panel, except for serving size designations where the panel is presented in a compact/simplified format.
- Set off by a heavy hairline (typically the heaviest bar in the panel) immediately below.

**Common failure:** Title is not the largest type, or not bolded, or appears as plain text without a bar.

---

## 2. Serving Size and Servings Per Container

Immediately below the title:

- **"Serving Size: [amount]"** — must be the maximum reasonable amount consumed at one eating occasion (e.g., 2 Gummies, 1 Capsule, 1 Tablespoon).
- **"Servings Per Container: [number]"** — must equal the container's total count ÷ serving size count.

Format example:
```
Serving Size  2 Gummies
Servings Per Container  30
```

Math check: serving size count × servings per container must equal total count declared on the PDP (e.g., 2 gummies/serving × 30 servings = 60 gummies, which must match "60 Gummies" on the PDP).

**Common failures:**
- Math mismatch between PDP count and panel.
- Serving size in unrealistic units (e.g., "1 gummy" when the dose requires 2).
- Missing servings per container.

---

## 3. Column Headers

Two columns:
1. (Left, unlabeled) — Ingredient names.
2. **"Amount Per Serving"** (center)
3. **"% Daily Value"** (right) — may be abbreviated as **"% DV"** if space-constrained.

Headers must be set off above the body of the panel by a heavy bar (lighter than the title bar but heavier than the inter-ingredient hairlines).

---

## 4. Order of Ingredients

Required order (21 CFR 101.36(b)(2)):

1. **Vitamins and minerals** — in the order specified in 21 CFR 101.9(c)(8)(iv) and 101.9(c)(9). The standard order is:
   - Vitamin A
   - Vitamin C
   - Vitamin D
   - Vitamin E
   - Vitamin K
   - Thiamin (B1)
   - Riboflavin (B2)
   - Niacin (B3)
   - Vitamin B6
   - Folate / Folic Acid (declare as "Folate" with mcg DFE; if from folic acid, also state "(as folic acid)")
   - Vitamin B12
   - Biotin
   - Pantothenic Acid (B5)
   - Choline
   - Calcium
   - Iron
   - Phosphorus
   - Iodine
   - Magnesium
   - Zinc
   - Selenium
   - Copper
   - Manganese
   - Chromium
   - Molybdenum
   - Chloride
   - Sodium
   - Potassium

2. **Other dietary ingredients** (botanicals, amino acids, fatty acids, enzymes, etc.) — listed below vitamins/minerals, in any order **except** that proprietary blends list components in descending order of predominance by weight within the blend.

3. **Proprietary blends** — declared with a total weight; components listed below in descending order of predominance by weight but without individual weights.

**Common failures:**
- Vitamins listed in alphabetical order rather than the order in 101.9.
- Botanical ingredients interspersed with vitamins instead of grouped below.
- Proprietary blend listed without total weight, or components not in descending order.

---

## 5. Amount Per Serving

The amount of each nutrient per serving, in metric units (mg, mcg, IU, g).

**Conventions:**
- Use the unit specified in the DV table (e.g., Vitamin A in mcg RAE, not IU — since 2016 NLEA update).
- Folate: declare in **mcg DFE** (Dietary Folate Equivalents).
- Vitamin D: declare in **mcg** (and IU optionally in parentheses) — since 2016 NLEA update.
- Vitamin E: declare in **mg** (since 2016).
- Round per 21 CFR 101.36(b)(3)(iv): generally to nearest whole increment for amounts ≥ 10, decimals for smaller amounts.

**Common failures:**
- IU used for Vitamin A, D, or E (post-2016 these should be in mcg/mg).
- Folate declared in mcg without DFE qualifier.
- Wildly precise decimals on small amounts (e.g., "0.327 mg") — round per the rounding rules.

---

## 6. % Daily Value

The % DV based on the established Daily Value for adults (or, if labeled for children 1–3, the child DV). DVs are in 21 CFR 101.9(c)(8)(iv).

- For nutrients with an established DV: show % DV in the right column.
- For nutrients without an established DV (most botanicals, amino acids, enzymes): use the **† symbol** in the % DV column.
- A single footnote at the bottom of the panel reads: **"† Daily Value not established."**

**Common failures:**
- % DV missing entirely.
- Botanicals showing a fabricated % DV (no DV exists — should be †).
- † symbol used but no corresponding footnote text.
- Footnote text missing the word "established" or worded incorrectly.

---

## 7. Heavy Bars

The Supplement Facts panel requires specific bar weights to visually separate sections. Approximate guidance (FDA Labeling Guide):

- **Heaviest bar (≈ 7 pt or 3 pt depending on panel size)**: Below the "Supplement Facts" title, separating it from the serving size block.
- **Medium bar (≈ 3 pt or 1.5 pt)**: Above and below the column headers ("Amount Per Serving" / "% Daily Value"), separating headers from the ingredient list.
- **Lightest hairlines**: Between rows of ingredients.
- **Bar below the panel body**: Separates the ingredient list from the footnote section.

Exact point sizes are not always measurable from PDF artwork, but the *presence and relative weight* of these bars is verifiable visually. If bars are missing or all the same weight, that's a Major finding.

---

## 8. Proprietary Blends — 21 CFR 101.36(c)

When ingredients are combined in a proprietary blend:

- Declare the blend with a single total weight (e.g., "Adaptogenic Blend: 500 mg").
- List the components of the blend immediately below, indented, in **descending order of predominance by weight**.
- Individual component weights are **not required**, but if any individual weight is disclosed, all individual weights must be disclosed (the "all or none" rule).
- The blend name should be descriptive (e.g., "Adaptogenic Blend" is fine; "Proprietary Blend" alone is permitted but discouraged from a marketing standpoint).

**Common failures:**
- No total weight for the blend.
- Components listed not in descending order (often hidden — the brand puts the marquee ingredient first regardless of weight).
- Disclosing weight of one component but not others.

---

## 9. Botanical Ingredients — 21 CFR 101.36(d), 101.4(h)

Botanicals must be identified by:

- **Common name** (e.g., "Turmeric")
- **Latin/scientific name in parentheses** (e.g., "(Curcuma longa)") — required when there is more than one common name or when the common name is ambiguous; best practice to always include.
- **Plant part used** — e.g., "(root)," "(leaf)," "(rhizome)," "(extract)" — required per 21 CFR 101.4(h)(1).
- **Standardization marker**, if standardized — e.g., "(standardized to 95% curcuminoids)" or "(standardized to 5% withanolides)."

Example of compliant entry:
> Turmeric (*Curcuma longa*) (root extract) (standardized to 95% curcuminoids) — 500 mg

**Common failures:**
- Latin name missing.
- Plant part missing (most common failure — labels often say "Turmeric Root" but omit Latin name, or "Curcuma longa" but omit "root").
- Marketing names used without botanical identification (e.g., "ProprietaryBlendX™" with no underlying botanical names).

---

## 10. Other Ingredients — 21 CFR 101.4(g)

Listed immediately below the Supplement Facts panel (outside the panel box), under the heading **"Other Ingredients:"**

- Includes all ingredients not declared inside the panel (carriers, excipients, flavors, colors, preservatives, sweeteners, anti-caking agents).
- Listed in **descending order of predominance by weight**.
- Use common or usual names.
- Specific color additives must be named (e.g., "FD&C Red No. 40," not "color").

---

## 11. Panel Format Options

Three layouts per 21 CFR 101.36(e)(1):

1. **Standard** — single column, full panel.
2. **Tabular** — when vertical space is limited, panel may be rotated 90° (less common).
3. **Linear** — for very small packages (≤ 12 sq in available), permitted only when standard format will not fit. Format: "Supplement Facts: Serving Size [x], Servings Per Container [y], [Ingredient] [amount] [%DV], ..." 21 CFR 101.36(i).

**Common failures:**
- Linear format used on a package that has room for the standard format (linear is a last resort, not a designer convenience).

---

## 12. Footnotes

Two footnotes commonly appear at the bottom of the panel:

1. **"† Daily Value not established."** — required if any † symbol appears in the % DV column.
2. **(Optional)** A general footnote about percent Daily Values being based on a 2,000-calorie diet — this is required for conventional food panels but **not** for Supplement Facts panels. Including it is not a violation but is not standard practice.

---

## Pass criteria for the Supplement Facts panel

| Sub-element | Pass criteria |
|---|---|
| Title | "Supplement Facts" — bold, largest type, heavy bar below |
| Serving size / Servings per container | Both present; math matches PDP count |
| Column headers | "Amount Per Serving" and "% Daily Value" present, set off by medium bar |
| Order of ingredients | Vitamins/minerals in 101.9 order, then other dietary ingredients |
| Amount per serving | Correct units (post-2016 NLEA); rounded correctly |
| % Daily Value | Present for nutrients with DV; † for those without; † footnote text matches "Daily Value not established." |
| Heavy bars | Title bar, header bar, and inter-row hairlines all visually distinguishable |
| Proprietary blends (if any) | Total weight declared; components in descending order; "all-or-none" weight rule observed |
| Botanicals | Common name + Latin name + plant part; standardization noted if applicable |
| Other Ingredients | Heading present; listed in descending order; colors specifically named |
| Format | Standard layout unless space justifies tabular/linear |
